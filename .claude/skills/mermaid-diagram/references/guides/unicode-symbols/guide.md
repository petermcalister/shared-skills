## Usage Tips

- Place symbols at the START of labels: `🔐 Auth Service`
- Use in subgraph titles: `subgraph "☁️ AWS Cloud"`
- Combine for clarity: `📨 📥 Message Queue (Incoming)`
- Keep consistent: same symbol = same concept throughout the diagram
