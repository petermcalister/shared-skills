# Theme → Palette Mapping

Maps story-present's 7 audience-inferred themes to pptx_builder's 12 palettes.

## Default mapping (used when `--palette` is not specified)

| Theme | Palette | Rationale |
|-------|---------|-----------|
| business | midnight-exec | Corporate dark navy + gold accent |
| tech | ocean | Dark + teal/green (closest to GitHub dark) |
| dark | midnight | Pure dark + blue/green |
| editorial | warm-terracotta | Warm neutrals matching serif register |
| vibrant | coral-energy | High-saturation orange for creative |
| minimal | charcoal | Clean grey for internal/standup |
| conference | ocean-gradient | Teal + cyan, code-friendly |
| consulting | midnight-exec | Same as business |
| status | teal-trust | Trust/stability green for RAG dashboards |
